Web Accessibility

웹이란 '장애에 구애 없이 모든 사람들이 손쉽게 정보를 공유할 수 있는 공간'으로 그 내용의
이해가 쉬워야 하며, 견고성을 지니고 충분한 웹 접근성을 확보할 수 있어야 합니다.

    • Perceivable 인식의 용이성

      명로성
      텍스트 콘텐츠의 명도 대비

      화면확대가 가능한 주요 콘텐츠의 타이틀과 아이콘의 명도대비는 3:1 이상을 만족합니다.

    • 텍스트와 아이콘의 명도 대비를 Colour Contrast Analyser 프로그램으로 테스트한 결과 캡쳐 이미지
    • Operable 운용의 용이성

      충분한 시간 제공
      정지 기능 제공

      자동재생 되어지는 상단 메인배너에 일시정지
      가능한 제어 기능을 추가하였습니다.

    • 메인배너에 일시정지 버튼과 페이지 넘김 제어가 가능한 버튼 캡쳐 이미지
    • Understandable 이해의 용이성

      가독성
      기본 언어 표시

      한국 웹 접속자들을 위한 사이트의 기본
      언어는 한국어를 기준으로 합니다.

    • HTML문서의 상단에 한국어를 기본으로 하는 마크업 캡쳐 이미지
