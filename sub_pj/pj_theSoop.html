Web Accessibility

웹이란 '장애에 구애 없이 모든 사람들이 손쉽게 정보를 공유할 수 있는 공간' 으로 그 내용의 이해가 쉬워야 하며, 견고성을 지니고 충분한 웹 접근성을 확보할 수 있어야 합니다.

    • Perceivable 인식의 용이성

      명료성
      텍스트 콘텐츠의 명도 대비

      화면확대가 가능한 주요 콘텐츠의 타이틀과 아이콘의 명도대비는 3:1 이상을 만족합니다.

    • 텍스트와 아이콘의 명도 대비를 Colour Contrast Analyser 프로그램으로 테스트한 결과 캡쳐 이미지
    • Operable 운용의 용이성

      충분한 시간 제공
      정지 기능 제공

      자동재생 되어지는 상단 메인배너에 일시정지
      가능한 제어 기능을 추가하였습니다.

    • 자동재생 배너에 일시정지, 재생 기능 추가 스크립트 코드 
                          
                      //자동재생 배너 일시정지 버튼 기능 추가
                      $('.main_top .controller .autostop').on('click', function () {
                          $(this).hide().siblings().show();
                          mySwiper1.autoplay.stop();
                          return false;
                      });
                      //자동재생 배너 플레이 버튼 기능 추가
                      $('.main_top .controller .autoplay').on('click', function () {
                          $(this).hide().siblings().show();
                          mySwiper1.autoplay.start();
                          return false;
                      });
